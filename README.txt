Important:
Add your logo image into this folder and name it:
server-icon.png

To update the Roblox game link:
Open index.html and replace https://www.roblox.com with your real Roblox game URL.

To update the launch countdown:
Open script.js and change this line:
const launchDate = new Date("2026-12-31T18:00:00").getTime();
